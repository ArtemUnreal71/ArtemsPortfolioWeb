Fibretel

Web Server udělaný za pomoci frameworku ASP.NET pro firmu Fibratel.

Tento web byl mou první pracovní zkušeností ve sféře IT. Web jsem dělal pro firmu Fibretel - dostal jsem se k ní přes kamaráda. Po dokončení webu se ale s partnery dohodli, že web nepotřebují, takže mi tento projekt zůstal jako takový Template pro další weby. Každopádně, rozhodl jsem se ho přidat do této sekce, jelikož jsem přeci jen měl nějaké určité požadavky a musel jsem brát ohled na spoustu podmínek aby web fungoval i při reálné situaci, a ne jen u mě na localhostu. Jelikož ale web nebyl zprovozněn, tak stále nemám zkušenosti s hostingem, ssl certifikáty, správnými adresami atd.
